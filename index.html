What is CSS?

CSS stands for Cascading Style Sheets.CSS describes how HTML elements are to be displayed on screen, paper, or in other media.CSS saves a lot of work. It can control the layout of multiple web pages all at once. Cascading refers to like waterfall. It checks the logic from sequencially from top to bottom and implement the last logic of a property given. Thats why it is called cascading.

Types of CSS

  1. Inline: CSS is written in the same line of the html. Here style is working as attribute.h1 Style="Property:Value"
  2. Internal: CSS is written in the same page of the html. Here style is working as element. Need to select the line tag with selector.Style should be in after title within head tag. i.e: For h1:h1{"Property:Value"}
  3. External: CSS is written in separate file which saved as css. Need to link with the html file; using link tag and rel & href attribute.Totally different from previous two types css. It is mostly used.

Sequence of the power of the css type is like : Inline>Internal>External

Selector Power sequence like: ID>Class>Tag.

ID: an identifier unique within the document.
Class:an identifier that can annotate multiple elements in a document.
